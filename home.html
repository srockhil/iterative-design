Introduction

In this project we set out to learn about iterative design, and how this process of prototyping, user-testing and analyzing is essential to the development of a good user interface. To do this, we undertook the task of designing an interface for a startup: Ribbon. We started by reading about the company to gain insight into their purpose and important features they offer. Using this knowledge we made initial sketches and used them to create lo-fi wireframes of our design. From there we created our first hi-fi mockup. This mockup was critiqued by our classmates, and we used the feedback to improve our interface before submitting it to a user testing site. Finally, we analyzed the results of 3 user questionnaires and video-responses.

The Startup: Ribbon

Ribbon gives creators and businesses a way to sell tickets for online virtual events. The website facilitates everything from event creation to ticket sales. Users can create weekly, monthly or any type of subscription to events. Their zoom integration automatically creates meetings with unique links and passwords for each event. The site allows users to manage and edit their events, see event participants, and quickly invite new event participants.

Contacting the Startup

Email Body:
For an iterative design project, my classmates and I created hi-fi desktop mockup of an interface for Ribbon! We are students at Brown University taking a UIUX course, and we were given the task of designing a user interface for a startup of our choice. We read about your startup and designed an interface for a creator portal based on the information on ribbonexperiences.com. After receiving critiques from fellow students and incorporating solutions, we submitted the mockup to usertesting.com! The main issue we ran into was creating a “Create Event” form that completely encompassed the features offered by Ribbon — we solved this by separating the form into multiple sections, and using design principles to facilitate easy navigation. The mockup can be found at https://xd.adobe.com/view/f56c1a51-4754-41a6-8593-e4c7a9868aa4-a03f/ . If you have any time, we would love to know what you think of our design! Thanks so much!
